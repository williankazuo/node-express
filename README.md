# node-express

## Init
- ```npm init -y```
- ```npx tsc --init```


### Dev dependencies
Typescript
```
npm install typescript -D
```

TSX CLI to run typescript code:
```
npm install tsx -D
```

tsup CLI to build
```
npm install tsup -D
```

Vitest test frameworks, simpler than jest, faster
```
npm install vitest -D
```



## Commands
Start app
```
npm run start
```

Start app in watch mode
```
npm run start:dev
```
 
Build the app
```
npm run build
```

Run Tests
```
npm run test
```
